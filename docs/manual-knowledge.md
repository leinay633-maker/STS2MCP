# STS2 人工知识库

## 当前总纲 - 铁甲战士稳定路线

不要把“完美打击+ + 多打击牌”理解成唯一终局路线。低阶和第一幕里，额外打击、剑柄打击、双重打击、打击木偶、地狱狂徒、双截棍等协同会让完美打击成为很强的前期输出外壳；但从第二幕开始，长期更稳的方向是转向“消耗 + 抽牌 + 能量 + 易伤爆发”。

后续决策优先级：
- 优先补燃烧契约、祭品、放血、剑柄打击、恶魔之焰、黑暗之拥、邪眼、坚毅+、重振精神等能处理坏手牌或把消耗转成收益的组件。
- 完美打击+仍可作为高伤害牌使用，但不要为了它无脑保留所有基础打击；如果没有打击木偶、地狱狂徒、幽灵种子等强协同，第二幕以后可以开始删除低质量基础打击。
- 删除优先级通常是诅咒/受伤 > 低质量防御 > 多余基础打击。若本局有巨大扭蛋、打击木偶、地狱狂徒或完美打击已经升级，则基础打击的删除优先级下降。
- 恶魔之焰/原始力量这类会改变整手牌的牌，必须在打出完美打击、关键攻击或确认斩杀线之后再用，避免误伤核心输出。
- Boss 和精英战中，易伤窗口、双截棍/开心小花计数、祭品/放血的血量成本都要按最新 GET 精算；关键回合继续保持“一次操作后重新 GET”。

## 2026-05-01 - 铁甲战士 A1 三幕 Boss 通过

结果：铁甲战士 A1 到达 50 层，击败三幕双 Boss，随后进入建筑师结算覆盖层 `NGameOverScreen`。以后把这条记录视为“三幕 Boss 已打过/本局已结束”，不要当成仍在战斗中。

最终构筑：
- 核心输出：完美打击体系，大量保留名字里带“打击”的牌。
- 核心加速：地狱狂徒+ x2。抽到打击牌会自动打出，抽牌本身就能转化为随机伤害。
- 关键辅助：战斗专注+ x2、祭品+、震荡波+、凶恶+、燃烧+、旋风斩+、壁垒+、火焰屏障+ x2。
- 关键遗物：打击木偶非常契合完美打击和所有打击牌；金刚杵、开心小花提高爆发回合；怀表适合低出牌数的准备回合。

牌组经验：
- 完美打击在“打击牌数量足够多”的情况下可以作为主轴。体系成型后，不要无脑删太多打击。
- 地狱狂徒+ 对这个体系提升很大，因为抽牌会变成免费攻击。
- 抽牌很重要，但要注意顺序：战斗专注会禁止本回合后续抽牌，能先用其他抽牌/检索时不要先打它。
- 震荡波+、上勾拳、战栗、熔融之拳都很关键；易伤会把打击体系从“能打”放大到“能斩杀”。
- 壁垒+ 不是必须，但配合火焰屏障+ 能稳住长线 Boss 战，把多余格挡留到坏手牌回合。

Boss 经验 - 三幕双 Boss 碾碎爪/火箭：
- 两个 Boss 都有“蟹怒”：一个死后，另一个获得 +6 力量和 99 格挡。
- 理想打法是先把两边血量都压低，再击杀其中一个；如果幸存者已经低血且我方血量/格挡很稳，也可以先杀一个。
- 本局先杀火箭，碾碎爪获得 +6 力量和 99 格挡；靠火焰屏障+/壁垒稳住，最后用旋风斩+结束。
- 被包围/背刺机制下，朝向很重要。优先面对当前高攻击的一侧。
- 三幕 Boss 前后药水不要省。能量药水帮助关键启动，瓶装潜能修正了后期坏手牌。

MCP 操作经验：
- 重要出牌或用药后要重新读取状态。Boss 战中动作后可能有 2-3 秒状态延迟。
- 手牌索引会在动画和出牌后变化，关键牌不要沿用旧索引，先重新查询。
- `combat_select_card` 参数是 `card_index`，不是 `index`。
- `NGameOverScreen` 是 MCP 未处理的覆盖层，可能需要手动在游戏里点结算按钮。

未来提醒：
- Boss 战前提醒用户用 STS2Saves 创建或确认快照。
- 除非用户明确要求，不启用 watchdog/自动循环。每次由用户启动，并用“继续”手动授权下一步。

## 2026-05-01 - 铁甲战士 A2 第一幕 Boss 通过

结果：A2 第一幕击败 Boss 仪式兽，战后 54/87，拿到 100 金、力量药水，Boss 稀有牌选择恶魔之焰。当前牌组 22 张，药水为力量药水 + 再生药水。

本幕关键资源：
- Neow 选择巨大扭蛋，额外拿到风箱 + 烛台，同时牌组多了一组打击/防御。
- 风箱会升级每场战斗的起手牌，配合灯笼第一回合和烛台第二回合，第一幕非常适合主动抢伤害。
- 缩放仪让 Boss 前可以优先敲牌而不是休息。本局最终营火敲了燃烧+，Boss 战收益明显。

牌组经验：
- 完美打击+ 在巨大扭蛋多给一张打击后依然很强，但牌组会变厚，后续要优先补抽牌、检索、删防御/普通打击，而不是继续无脑加牌。
- 燃烧+、主宰、预备打击提供的力量能把双重打击、剑柄打击、御血术、完美打击一起抬高，A2 第一幕输出足够。
- 狂怒+ 在多攻击回合能补很多格挡；如果敌人没有高威胁，先打狂怒再连续攻击，比单纯防御更好。
- 恶魔之焰适合这套厚牌组：把一手基础牌转成爆发伤害，消耗牌也能提高灰烬打击后续伤害。使用时要确认能杀或能承受下回合，因为它会清空手牌。

Boss 经验 - 第一幕仪式兽：
- 仪式兽在血量过 150 附近会触发一次硬直/中断，本局用液态记忆捞回御血术+，把 Boss 从 150 以上打到 129，直接取消了当回合 18 攻击和强化。
- Boss 后半会施加“昏眩”：本回合只能出 1 张牌。昏眩回合不要按普通斩杀线计算，要优先打一张最高收益牌。本局用剑柄打击+压血并过牌，下一回合再双重打击 + 打击斩杀。
- Boss 药水不要省。清晰药水用于第一回合扩展手牌，液态记忆用于跨过阈值，比留到第二幕更赚。
- 再生药水在安全血量时可以保留。本局 66 血吃一口 18 后仍安全，保留再生药水进入第二幕。

路线/操作经验：
- A2 第一幕可以吃精英，但要在 Boss 前保留至少一个强药水或有缩放仪/营火保障。
- MCP 奖励动作用 `claim_reward` 时字段是 `index`，不是 `reward_index`；奖励会左移，领取后要重新读取状态。
- Boss 战动作延迟明显，尤其是阈值、昏眩、抽牌后，关键判断前必须重新 GET 一次。

## 2026-05-01 - 铁甲战士 A2 第二幕 Boss 通过

结果：A2 第二幕击败 Boss 知识恶魔，进入 A3 地图入口。战后 38/87，金币 130，保留鲜血药水，Boss 稀有牌选择黑暗之拥。当前牌组 31 张。

本幕关键资源：
- 商店买王室印章并附魔恶魔之焰，让恶魔之焰固有 + 保留。这个改动是第二幕最关键的强度来源，起手能稳定把一手牌转成爆发伤害，并快速堆消耗牌数量。
- 商店补了永恒铠甲、熔融之拳，并删了一张防御；后续假商人买了打击木偶？？？、锚？？？、开心小花？？？。打击木偶继续强化完美打击/剑柄打击/普通打击，锚提供起手安全垫。
- 遗物红面具降低前期伤害压力，钢笔尖把完美打击+变成主要爆发窗口，古茶具套装让 Boss 战第一回合能量足够展开。
- 第二幕最后营火选择敲撕裂+，思路正确：牌组已有血墙+、御血术、怨恨+，自伤转力量可以提高长线 Boss 战上限。

牌组经验：
- 王室恶魔之焰和黑暗之拥非常契合。恶魔之焰、战鼓、熔融之拳、主宰都会消耗；黑暗之拥能把这些消耗转成抽牌，比狂宴和扯碎更适合当前厚牌组。
- 燃烧+、力量药水、预备打击、主宰提供的力量，会被易伤和钢笔尖进一步放大。Boss 战里钢笔尖给完美打击+打出 144 点，是斩杀核心。
- 熔炉的祝福适合留给“多张未升级攻击 + 关键基础牌”的 Boss 回合。本局在痛击、剑柄打击、防御、多个打击同手时使用，收益高于继续贪保留。
- 战鼓在缺防御、需要找关键牌时值得打，但之后每回合会消耗抽牌堆顶。已经有黑暗之拥后，战鼓的负面有机会转成额外抽牌。

Boss 经验 - 第二幕知识恶魔：
- 知识恶魔会给二选一负面。本局选择“瓦解：每回合结束受到 6 点伤害”，没有选“心灵腐化：每回合少抽 1 张”。对厚牌组和需要找爆发牌的铁甲来说，少抽比固定掉血更危险。
- 选择瓦解后必须加速击杀，不能长期防守。本局用力量药水 + 燃烧+ + 易伤 + 钢笔尖，把 Boss 从 379 血压到 4 回合内击杀。
- 第一回合用恶魔之焰消耗防御、狂怒等牌，既打出爆发，也提高灰烬打击后续伤害。恶魔之焰要确认本回合收益足够，因为它会清空手牌。
- Boss 第二回合打 17 时没有足够格挡，选择继续铺力量和上易伤，血量从满血降到 64。这个选择成立是因为有缩放仪进场满血、鲜血药水兜底、Boss 已经进入可斩杀节奏。
- Boss 后半 8x3 攻击很疼，但如果 Boss 已经 100 血左右，可以继续压伤害。鲜血药水可以留到下一回合再判断，不必在 64 血提前喝。

怪物/事件经验：
- Decimillipede 多段虫有复活机制：如果没有在接续窗口内清完，已死段会复活。以后打这类敌人要把所有段都压到可同回合击杀，不要只逐个击杀。
- Entomancer 的 Personal Hive 会让每次攻击往抽牌堆塞 Dazed。打它时少用低收益多段攻击，优先用高单次伤害、药水和恶魔之焰快速结束。
- 假商人可以买遗物，不会因为正常购买直接开战；只有特殊投掷/攻击类选项才需要警惕。本局买假打击木偶、假锚、假小花都有效。

操作经验：
- 宝箱房必须先用领取遗物动作拿遗物，再 `proceed`。本局宝箱房看到 Eternal Feather 后误用 `proceed`，导致遗物没有加入列表；以后宝箱结算前必须重新 GET 确认遗物已进入 relics。
- 多张牌连续出时，手牌索引会立刻左移。关键牌不要假设旧索引仍然指向同一张牌；尤其是 Boss 战里要么逐张 GET，要么按左移后的新索引严格计算。
- 休息处动作是 `choose_rest_option`。需要回血时选 Rest，需要升级时选 Smith；Boss 前如果有缩放仪/血量足够，可以优先敲关键牌。

## 2026-05-01 - 铁甲战士 A2 三幕 Boss 通过

结果：A2 三幕击败 Boss 门扉缔造者。Boss 被击杀时战斗结算血量为 83/88，随后进入建筑师事件并切到 `NGameOverScreen` 覆盖层。MCP 在覆盖层里显示 hp=0，这是结算/游戏结束界面字段，不要误记为战斗内死亡。

三幕关键资源：
- 三幕商店因领主阳伞自动获得全部货物，关键新增为硫磺、药水腰带、皇家枕头，以及战鼓、战斗专注、亮剑、秘密技法等补牌。硫磺让 Boss 战变成力量竞速，但我方力量成长更快。
- 商店移除腐朽后，卡牌移除 UI 卡住，又移除了一张基础防御才退出；以后商店移除后要重新 GET 确认界面状态。
- Boss 前营火没有休息，选择敲永恒铠甲+。因为缩放仪会在 Boss 开局回满 25 血，本局进 Boss 时满血 88/88。
- Boss 前提醒用户创建/确认 STS2Saves 快照是必要流程；只需要 Boss 前提醒，不需要每个节点都提醒。

Boss 经验 - 门扉缔造者：
- 开局门扉先是 999999999 血的召唤形态，随后变成 489 血真身。第一回合不要急着把输出交在门形态上，优先铺长期能力。
- 能力药水三选一时选择壁垒，优先级高于岩石铠甲和黑暗之拥。壁垒配合固化药水、覆甲、血墙可以把防守压力一次性解决。
- 第一回合铺壁垒、战鼓、杂耍，再打防御+存格挡。恶魔之焰+继续保留，不在手牌质量高时乱烧。
- 真身第一轮 489 血、打 31。本局用狂怒+配合多攻击补格挡，再用灰烬打击打出高额伤害，把 Boss 从 489 压到 300 左右。灰烬打击在消耗堆变大后是核心输出，不只是普通一费攻击。
- 固化药水在已有壁垒时非常强。本局 19 格挡面对 26 攻击时使用，直接变成 57 格挡，后续几轮都靠这层安全垫打输出。
- 秘密武器在能量紧张时优先找 0 费攻击。本局选择愤怒，配合闪电霹雳+、御血术自伤、怨恨+三连，把 Boss 从 297 压到 89。
- 撕裂 + 自伤牌 + 怨恨+ 是 A2 后期铁甲的关键爆发线。祭品+、放血、御血术既能触发撕裂，也能让怨恨+变成三段攻击。
- 终局 Boss 剩 86 血时，钢笔尖计数为 8。正确斩杀线是放血补净能量，然后完美打击+作为第 9 张攻击，熔融之拳作为第 10 张攻击吃钢笔尖双倍，稳定击杀。

牌组经验：
- 王室认证恶魔之焰+这局主要作为保留的终结按钮，而不是默认起手烧牌。Boss 真身战中手牌质量高、能靠钢笔尖和灰烬打击斩杀时，不要强行赌恶魔之焰。
- 硫磺会让敌人力量持续增加，不能无限拖；但有壁垒和固化药水时，可以先把防守垫高，再集中用易伤、怨恨+、灰烬打击、完美打击+抢节奏。
- 战鼓会每回合消耗抽牌堆顶。已有黑暗之拥时它可能转化为抽牌收益；没有黑暗之拥在场时，要注意它会吃掉关键牌。本局 Boss 第一回合战鼓叠到 2 层，整体收益仍为正。
- 祭品+带灵魂之力、不消耗，是 Boss 战中最重要的续航引擎之一。后期看到祭品+通常优先打，用生命换能量和抽牌，再靠燃烧之血/药水兜底。

MCP 操作经验：
- `card_select` 里的 `select_card` 参数是 `index`，不是 `card_index`。本局秘密武器选愤怒时先传错字段，界面没有变化；改成 `{ action='select_card'; index=2 }` 才成功。
- 选择卡牌奖励仍然是 `select_card_reward` + `card_index`；战斗中手牌选择仍然是 `combat_select_card` + `card_index`。三个选择接口字段不同，不能混用。
- AOE 卡有时带 target 会出现不完整结算；杀灭本局表现不稳定，不要围绕它算精确斩杀。关键斩杀优先用单体目标明确的攻击和钢笔尖计数。
- Boss 的“紧握”会让手牌描述追加“失去 1 点能量”。计算终局斩杀时要把这当成额外能量成本；祭品+、放血在这种情况下通常是净 +1 能量。
- Boss 死后奖励为空但 `can_proceed=true`，需要 `proceed` 进入建筑师事件；建筑师只有“威胁”一个选项，点完后进入 `NGameOverScreen` 覆盖层，可能需要用户在游戏内手动点结算按钮。
